Définitions

L'Application
Désigne l'application mobile « Nextation ».
L'Éditeur
Désigne le responsable de l'application, identifié ci-dessous, contactable à support@studio.bzh.
L'Utilisateur
Toute personne physique utilisant l'Application.
Token FCM
Identifiant technique pseudonymisé, unique à l'installation de l'instance de l'application, permettant l'acheminement des notifications push via les serveurs Firebase.
Données de Transport
Informations techniques nécessaires au service d'alerte, telles que le numéro du train (ex : TGV 8600), les codes UIC des gares (ex : 87391003 pour Paris-Montparnasse), la date et la plage horaire de surveillance.
Backend (TrainsFlow)
Infrastructure technique et logique (Google Cloud / Firebase) traitant les alertes et les notifications.
OS
Système d'exploitation mobile de l'appareil de l'Utilisateur (ex : Android).
